# 前端支持聊天文件上传

**任务时间**: 2026-03-24 20:30

## 任务目标

修改前端聊天界面和逻辑，支持用户在发送消息时上传 Excel 和 CSV 文件，对接后端已优化的聊天接口。

## 需求分析

1.  **UI 变更**
    *   在聊天输入框附近添加“上传文件”按钮（图标）。
    *   支持选择文件（.xlsx, .xls, .csv）。
    *   显示已选择但未发送的文件列表（支持删除）。
    *   发送消息时，将文件一同发送。
    *   在聊天记录中显示已上传的文件信息（可选，视后端返回和需求而定）。

2.  **逻辑变更**
    *   修改发送消息的 API 调用，从 JSON 格式改为 `multipart/form-data`。
    *   处理文件选择、校验（大小、类型）逻辑。
    *   处理后端返回的文件信息。

## 设计方案

### 1. 组件修改
*   **ChatInput 组件** (假设存在): 添加文件选择 input (hidden) 和触发按钮。
*   **FileList 组件**: 显示待发送的文件预览。

### 2. API 修改
*   更新 API 请求函数，支持 `FormData`。

### 3. 校验
*   前端预校验文件类型和大小（10MB）。
