# 优化聊天接口支持文件上传

**任务时间**: 2026-03-24 20:12

## 任务目标

优化后端聊天接口，支持上传Excel和CSV文件，并实现安全管控机制。

## 需求分析

1. **文件上传支持**
   - 支持Excel文件（.xlsx, .xls）
   - 支持CSV文件（.csv）
   - 文件随聊天消息一起提交

2. **安全管控要求**
   - 文件类型白名单：仅允许.xlsx, .xls, .csv
   - 文件大小限制
   - 文件内容验证（魔数检测）
   - 文件名安全检查
   - 病毒扫描（可选）
   - 临时文件清理机制

## 设计方案

### 1. 安全策略设计

#### 文件类型验证
- **扩展名白名单**: `.xlsx`, `.xls`, `.csv`
- **MIME类型验证**: 
  - Excel: `application/vnd.openxmlformats-officedocument.spreadsheetml.sheet`, `application/vnd.ms-excel`
  - CSV: `text/csv`, `application/csv`
- **魔数（Magic Number）验证**: 检查文件头部字节确认真实文件类型

#### 文件大小限制
- 单文件最大: 10MB（可配置）
- 防止DoS攻击

#### 文件名安全
- 禁止路径穿越字符: `..`, `/`, `\`
- 文件名长度限制: 最大255字符
- 使用UUID重命名存储文件

#### 存储安全
- 临时存储目录隔离
- 文件权限控制（只读）
- 定期清理过期文件

### 2. API设计

#### 接口路径
`POST /api/chat/message`

#### 请求格式
- Content-Type: `multipart/form-data`
- 字段:
  - `message`: 聊天消息文本
  - `session_id`: 会话ID
  - `files`: 文件列表（可选，支持多文件）

#### 响应格式
```json
{
  "message_id": "uuid",
  "response": "AI回复内容",
  "uploaded_files": [
    {
      "original_name": "data.xlsx",
      "file_id": "uuid",
      "size": 12345,
      "type": "excel"
    }
  ]
}
```

### 3. 配置项

在配置文件中添加：
```yaml
file_upload:
  enabled: true
  max_file_size: 10485760  # 10MB
  allowed_extensions:
    - .xlsx
    - .xls
    - .csv
  allowed_mime_types:
    - application/vnd.openxmlformats-officedocument.spreadsheetml.sheet
    - application/vnd.ms-excel
    - text/csv
    - application/csv
  temp_dir: /tmp/datalake_uploads
  cleanup_after_hours: 24
```
